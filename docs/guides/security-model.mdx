title: Ory's Security Model
---

Ory works with a novel security model which might take a moment to get used to. Compared to other solutions,
or manages the identity's session for you whenever possible, and has best-in-class protections for browser vulnerabilities (XSS, CSRF)
built in.

Ory makes use of HTTP cookies to store an identity's session whenever the client is a browser. Getting cookies to work
cross domains can be challenging. This document walks you through the concepts neccessary
to understand how this model works.
